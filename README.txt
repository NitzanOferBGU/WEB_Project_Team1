# GetProject - בניית מערכות מבוססות WEB (חלק ג')

**מגישים:** איתי מזרחי, ניצן עופר ואיתי שרם.

---

## תיאור הפרויקט
GetProject היא מערכת רשת לניהול פרויקטים וגיוס משתתפים. המערכת מאפשרת למשתמשים מסוג "מוביל" לפתוח פרויקטים חדשים (כולל תיאור, תאריכים, תכונות נדרשות והעלאת תמונה), ולמשתמשים מסוג "משתתף" להגיש בקשות הצטרפות לפרויקטים אלו. המוביל יכול לנהל את הפרויקטים שלו (עריכה ומחיקה) וכן לאשר או לדחות את המועמדים לפרויקט.

הפרויקט נכתב בארכיטקטורת צד-לקוח וצד-שרת מופרדת, עם דגש על מודולריות, אבטחת מידע (ולידציות בצד השרת) וניהול מסד נתונים רלציוני באופן מלא.

---

## דרישות קדם (Prerequisites)
על מנת להריץ את הפרויקט על מחשבך, יש לוודא שמותקנים:
1. Node.js
2. שרת MySQL מקומי (לדוגמה דרך MySQL Workbench, XAMPP או WAMP).

---

## הוראות התקנה והרצה (לבדיקת המטלה)

### 1. התקנת תלויות (Dependencies)
הרץ את הפקודה הבאה בטרמינל של גיט:
npm i

### 2. הקמת מסד הנתונים
בתוך תיקיית db בפרויקט נמצאים סקריפטים מוכנים מראש של SQL:
* הרץ את הסקריפט init-db.sql במערכת ה-MySQL שלך. 
* הפקודה תיצור באופן אוטומטי את מסד הנתונים GetProjectDB ואת כל טבלאות המערכת הנדרשות (Users, Projects, applications) עם הקשרים המתאימים.

### 3. חיבור השרת למסד הנתונים
כדי שהשרת יתחבר בהצלחה למסד הנתונים המקומי שלך:
1. גש לקובץ ההגדרות בנתיב: config/db.config.js.
2. עדכן את השדות USER ו-PASSWORD לפרטי ההתחברות של מסד הנתונים אצלך.

### 4. הפעלת השרת
בשורת הפקודה, הרץ את שרת ה-Node בעזרת הפקודה:
node index.js
או:
npm start
(תתקבל הודעה בקונסול: "השרת רץ על פורט 3000" וכן "מחובר למסד הנתונים").

### 5. כניסה לאתר
פתח את הדפדפן וגלוש לכתובת:
http://localhost:3000
